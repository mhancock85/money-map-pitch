# Money Map Pitch Deck

## Overview
This is a web-based pitch deck for the Money Map MVP, designed for **Burning Heroes** and **Wise** presentations.
It uses the same "Finazze" design language as the main website.

## How to View
1. Open `index.html` in your browser.
   - On Mac: Double click the file in Finder or run `open index.html` in terminal.
2. Use **Arrow Keys** (Up/Down or Left/Right) to navigate between slides.

## Structure
- **Slide 1**: Title & Hook
- **Slide 2**: The Problem (Fragmented Finances)
- **Slide 3**: The Solution (AI "Set and Forget")
- **Slide 4**: Business Model (SaaS for Money Coaches)
- **Slide 5**: Call to Action (Join Us)

## Customization
- Edit `index.html` to change text.
- Styles are handled in `style.css` (base) and `index.html` (slide specifics).
